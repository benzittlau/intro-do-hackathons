Tools

Libraries

Many languages have large ecosystems of libraries which you can leverage in your projects.

Using libraries can save you from spending time on things you're not learning.

Sometimes doing boring things the hard way is the best way to learn.

Libraries are critical to help you get an impressive result in a short time.

You should practice with the libraries you plan on using in advance, or could get slowed down with trivial issues.

Version Control

Tools for managing and merging your code, such as Git and GitHub, Hg and BitBucket.

These tools are often complicated and will slow down your initial speed, spending time learning tools instead of coding.

The benefits that version control brings to collaborating in a common code base are huge for teams larger than 2 members.

Collaboration

Learning with others can be incredibly helpful

Either pair with someone who knows what you want to learn

Or pair with someone who wants to learn the same things

Working with someone else adds overhead

In order to team up on a project you'll need to compromise on what you work on

Pairing with someone with complementary skills can be very powerful
